# javascript-web-datatable-button

## Description
A demo of using datatable with in memory
data source to illustrate CRUD operations.

## Tech stack
- jquery
- bootstrap

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
